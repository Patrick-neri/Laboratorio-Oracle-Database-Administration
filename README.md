# Laboratório-Oracle-Database-Administration
Repositório com laboratórios simulando situações reais do dia a dia de um DBA Jr

---

## Sobre este repositório

Este repositório documenta labs práticos que cobrem as responsabilidades do dia a dia de um DBA júnior, com ênfase em:

- Gestão e manutenção de estruturas lógicas e físicas do banco de dados
- Administração de usuários e permissões
- Configuração e monitoramento de componentes críticos (redo logs, control files, undo)
- Backup e Recovery
- Automação de verificações de saúde da instância via scripts SQL

---

| Componente       | Configuração                          |
